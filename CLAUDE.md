# CLAUDE.md

Fully pre-rendered static marketing site for SwissPlant GmbH, built with SvelteKit.

## Commands

```bash
node --run start   # dev server
node --run build   # production build → build/
node --run preview # production server
```

## Rules

- **Think end-to-end before acting.** Before starting any task, reason through the full approach from start to finish — what needs to change, how the pieces fit together, what could break, and what the finished result looks like. Don't begin editing until you have a complete picture.
- **Ask, don't assume.** Whenever any uncertainty arises — ambiguous requirements, multiple viable approaches, unclear scope, or missing context — stop and use `AskUserQuestion` to resolve it. Proactively surface uncertainties as soon as you notice them rather than guessing and proceeding. A clarifying question is always cheaper than reworking a wrong assumption.
